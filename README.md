# Ham Radio Log Utils

## Intentions:
This repository is intended to be a Python package that will contain some utility classes that can be used for working
with amateur radio QSO logs in both ADIF and Cabrillo format. 

## Current State:
Currently this project is in the very early phases of development. I am working on implementing ADIF support first, and
then will move to Cabrillo support afterwards, following that with support for converting between the two formats.
